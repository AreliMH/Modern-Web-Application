What's my research about ?

The goal of my research is to build a modern, responsive web application. In order to accomplish this, I needed an understanding of software engineering, scientific knowledge, and the tooling process. In an effort to approach the project I develop and iterate web application prototypes using modern web development technologies. As well as researching existing web applications for software engineering research labs to derive site requirements and architecture. And lastly by understanding how software development teams utilize software repositories.

Steps I've taken to approach the research

GitHub

I had to first understand how software development teams utilize software repositories. And what made this possible was to become familiar with the Git repository pipeline process and architecture. After acquiring the technique of GitHub the commands such as Cloning/forking repositories, creating new repositories, and committing software updates became second nature.
